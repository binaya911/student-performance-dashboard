# Student Performance Dashboard

A Python Dash web dashboard built to analyse and visualise student academic performance data.

## What it does
- Visualises relationships between study hours, attendance, and exam scores
- Shows pass/fail distribution across student demographics
- Interactive filters by gender, parental education, and internet access

## Tech used
Python, Dash, Plotly, Pandas

## How to run
pip install -r requirements.txt
python "main dash.py"
Then open http://127.0.0.1:8050 in your browser
